## 📦 Установка

### Шаг 1 — Скачать

**Через ZIP (рекомендуется):**
1. Зайди в [Releases](https://github.com/oinktechltd/freepass/releases)
2. Скачай `freepass-v2.zip`
3. Распакуй в удобную папку, например:
   - Windows: `C:\FreePass\`
   - macOS/Linux: `~/freepass/`

**Через git:**
```bash
git clone https://github.com/oinktechltd/freepass.git
cd freepass
```

---

### Шаг 2 — Включить режим разработчика

1. Открой Chrome
2. В адресной строке набери: `chrome://extensions`
3. В правом верхнем углу включи переключатель **«Режим разработчика»**

---

### Шаг 3 — Загрузить расширение

1. Нажми **«Загрузить распакованное»** (Load unpacked)
2. Выбери папку где лежит `manifest.json`
3. Иконка 🔓 появится в панели расширений

> **Совет:** Закрепи иконку — нажми 🧩 в панели → найди FreePass → 📌

---

### Шаг 4 — Использование

```
Нажать 🔓 → Нажать ⚡ Подключиться → Ждать 10–30 секунд → Готово
```

Расширение автоматически:
1. Загружает свежий список из тысяч прокси
2. Тестирует серверы параллельно
3. Применяет первый рабочий
4. Запускает 60-минутный таймер
5. За 5 минут присылает уведомление
6. По истечении — отключает прокси

**После 60 минут** просто нажми «Подключиться» снова.

## 🏗 Архитектура

```
freepass/
├── manifest.json    # Chrome Extension Manifest V3
├── background.js    # Service Worker: поиск прокси, PAC, таймер, alarms
├── popup.html       # UI попапа
├── popup.js         # Логика: прогресс, таймер, состояния
└── icons/           # 6 PNG иконок (синие/зелёные)
```

### Источники прокси

```
ProxyScrape API  →  тысячи HTTP-прокси, обновляются каждую минуту
Proxifly (jsDelivr CDN)  →  GitHub-зеркало, не блокируется в РФ
TheSpeedX/PROXY-List  →  крупнейший GitHub-агрегатор
monosans/proxy-list  →  авто-обновление каждые 10 минут
jetkai/proxy-list  →  ещё один верифицированный источник
```

### Как работает PAC

```
Запрос → PAC-функция → домен в списке? → ДА → прокси → сайт ✅
                                        → НЕТ → напрямую ✅
```

Российские сайты (VK, Яндекс и т.д.) идут **напрямую** — без прокси, на максимальной скорости.

## ⚙️ Добавить свой прокси

Открой `background.js`, найди `PROXY_SOURCES` и добавь свой HTTP-прокси в `PROXY_LIST` (закомментированный массив в начале файла), или установи его как первый в очереди на проверку.

Для максимальной стабильности рекомендуется свой VPS с 3proxy или Squid.

## ❓ FAQ

**Q: Долго ищет прокси?**  
A: Обычно 10–30 секунд. Тестируется до 30 серверов параллельно. Если очень медленно — попробуй через несколько минут (публичные прокси иногда перегружены в пиковые часы).

**Q: Прокси нашёлся, но YouTube всё равно не работает?**  
A: Некоторые прокси проходят тест-запрос, но блокируют видеостримы. Нажми «Отключить» → «Подключиться» — расширение найдёт другой сервер.

**Q: Мои данные логируются?**  
A: FreePass не собирает ничего. Код открытый. Публичные прокси теоретически могут видеть трафик — не вводи пароли и используй только для просмотра.

**Q: Работает ли с Edge?**  
A: Да, Microsoft Edge поддерживает расширения Chrome.

**Q: Firefox?**  
A: Пока нет (Manifest V3 требует адаптации). В планах.
